# Agent Logs

This directory stores concise task logs for meaningful agent work.

## Filename

- `YYYY-MM-DD-short-task-name.md`

## Template

- Task title
- Date
- Scope
- Change classes
- Files changed
- Why
- Validation
- Waivers
- Risks/follow-up
- Memory updates
- Gates summary

Skip logs for trivial read-only exploration.
